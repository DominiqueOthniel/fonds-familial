Mode d’emploi – Application “Famille Tiwa Joseph”

Gestion du fonds familial : membres, épargnes, crédits, sessions, dons, caisse, cassation.

Installation et première utilisation

  • Installez via l’installateur : Famille Tiwa Joseph Setup 1.0.0.exe
  • Lancez l’app depuis le Bureau ou le Menu Démarrer.
  • Base de données : C:\Users\<Vous>\AppData\Roaming\Famille Tiwa Joseph\tontine.db
  • Pour réutiliser une base existante, copiez votre tontine.db à cet emplacement.

Connexion

  • Admin : admin@tontine.com / admin1234
  • Adjoint : adjoint@tontine.com / adjoint1234

Navigation

  • Dashboard : synthèse des fonds et activités.
  • Membres : ajout, modification, historique.
  • Mouvements / Épargnes : saisie et suivi (épargne, dépenses, etc.).
  • Crédits : octroi, remboursements, pénalités auto lors de nouvelle session.
  • Caisse : entrées/sorties, solde, historique.
  • Dépenses communes : gestion par catégories.
  • Dons : ajout, filtre, suppression (admin).
  • Sessions : créer, terminer, renommer, supprimer, mouvements, export PDF.
  • Cassation : simulation, exécution, PDF.

Opérations courantes

Membres

  • Ajouter : Membres → Nouveau
  • Modifier/Supprimer : depuis la liste des membres

Mouvements

  • Ajouter un mouvement : choisir le type (épargne, dépense, etc.), valider

Crédits

  • Accorder : Crédits → Nouveau
  • Rembourser : saisir le montant, valider

Caisse

  • Ajouter entrée/sortie : saisir, valider
  • Consulter : historique et solde

Dépenses communes

  • Ajouter/Modifier/Supprimer des dépenses globales

Dons

  • Ajouter : Dons → Nouveau don
  • Filtrer : sélection de catégorie
  • Supprimer (admin) : icône “poubelle”

Sessions & Cassation

  • Créer une session : applique automatiquement 20% de pénalités sur crédits non remboursés.
  • Terminer : clôture et préparation à la cassation.
  • Renommer : identifiez clairement vos sessions.
  • Supprimer : uniquement si non active.
  • Mouvements : vue dédiée pour la session sélectionnée.
  • Cassation : simuler, exécuter, exporter en PDF.

Exports PDF

  • Mouvements d’une session : Sessions → Exporter PDF (enregistre dans Downloads).
  • Autres exports (si disponibles) : Membre, Cassation.

Données & sauvegardes

  • Base SQLite : tontine.db
  • Sauvegarde : copier régulièrement le fichier vers un support sécurisé.
  • Restauration : remplacer le fichier dans le dossier AppData/Roaming/Famille Tiwa Joseph.

Dépannage

  • “Unable to open database” : vérifiez l’existence/permissions du fichier dans AppData/Roaming. Copiez votre tontine.db si besoin.
  • Export PDF non visible : vérifier Downloads, relancer l’app. Désactiver l’antivirus si verrouillage.
  • Icônes/Images : redémarrer l’app, vérifier les ressources.

Support : en cas de problème, notez la version de l’app, l’action réalisée, l’heure, et joignez si possible une copie de tontine.db.
